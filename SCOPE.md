# Scope

This resource list catalogues the **Israeli AI ecosystem** — both individual projects and the broader community/infrastructure around them.

## Format

- **Project sections** (AI Agents, Agent Skills, Finance, Healthcare, etc.) are rendered as compact tables: Project · Description · Stars.
- **Ecosystem sections** (Communities, Meetups, Facebook Groups, Government Bodies, Conferences, Inference Providers, Startups, Hebrew Language Resources) are rendered as grouped bullet lists.
- Author attribution lives in the commit history and the linked repos, not inline in the README.

## In scope

- AI agents and agent skills (individual or bundled) targeting Israeli services, data, or users
- Skills bundles and collections
- MCP (Model Context Protocol) servers for Israeli data sources, services, or workflows
- Plugins (e.g. Claude Code plugins) with an Israel focus
- Voice agents and conversational AI built for Israeli use cases
- AI-powered dashboards, tools, and utilities targeting Israeli data or users
- Hebrew-language AI models, tooling, and evaluations
- Israeli AI communities, organizations, and associations
- Websites and hosted services that are AI-first and Israel-focused
- Curated lists and indexes of the above

## Out of scope

- General technology projects, even if Israel-focused, that are not AI-related
- Non-AI open source projects by Israeli developers
- Generic AI projects without an Israeli angle (no Israeli data source, service, language, or user focus)
- Commercial SaaS products without an open / inspectable component, unless they represent notable community infrastructure (e.g. community sites)

When in doubt, the test is: *is this project both (a) AI-related and (b) meaningfully Israel-focused?* Both must be true for inclusion.
